# FLICKSTER Movie API Project
This is a movie discovery web application that fetches data from The Movie Database (TMDB) API. Users can explore popular movies, search for movies by title, filter movies by genre, and view movie details such as overviews, release dates, ratings, and more.

Features
Popular Movies Slider: Displays a sliding carousel of popular movies.

Search Functionality: Search for movies by title.

Genre Filtering: Filter movies by genre to discover different types of films.

Movie Details: View movie details such as overviews, release dates, ratings, and more.

Responsive Design: The application is fully responsive, ensuring a smooth user experience on all devices.

Light/Dark Theme Toggle: Switch between light and dark themes to personalize the viewing experience.

Interactive UI: A modern UI built with HTML, CSS, and JavaScript, providing an engaging and smooth browsing experience.

Tech Stack
Frontend: HTML, CSS, JavaScript

Libraries/Frameworks: Bootstrap (for styling), FontAwesome (for icons), and ScrollReveal (for animations)

API: TMDB API (The Movie Database) for fetching movie data

Theme: Light/Dark theme toggle using localStorage
