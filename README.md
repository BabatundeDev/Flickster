# Flickster 🎬

**Flickster** is a **TMDB-powered movie streaming platform** that displays movies with real-time data including titles, posters, overviews, genres, and release dates.

---

## 🔥 Features

- Real-time movie data from TMDB API  
- Search by movie title  
- Light/Dark theme toggle  
- Genre filter  
- Responsive and animated UI  
- Movie details: title, poster, overview, release date  

---

## 🛠️ Technology Used

- HTML  
- CSS  
- JavaScript  
- TMDB API  
- ScrollReveal (for animations)

---

## 🚀 What It Does

Flickster fetches data from the TMDB API and displays movies in a clean, responsive, and interactive interface. Users can search for any movie, filter by genre, and switch between light and dark themes.
